# Setup

<div className="subheading">Requirements</div>

React-Agenda is based on React.

It uses the Internalization API [Intl](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Intl) in order to validate locale and timezone parameters, to format the dates and convert the times to be shown in the agenda.

<div className="subheading">Getting started</div>

Install package:

```
npm install f-react-agenda
```

Import component:

```js
import { Agenda } from 'f-react-calendar';
```

Use React-Agenda it's easy. For example, to show a basic agenda with full week view and display a time range from 10:00 AM to 15:00 PM, use the below code:

```js
<Agenda
    startTime={600}
    endTime={900}
    firstDay={1}
    daysNumber={7}
    locale="en-US"
/>
```
